Development plan data collector

You can create data that meets the development plan data standard with this prototype. You can record all the different development plans a local planning authority (LPA) makes (including local plans and neighbourhood plans) as data.

For each plan you can capture the events that happen and any documents that are published online during the plan-making process.

The process for collecting local plans

  1. You enter data into this prototype
  2. We use this data to seed a dataset of development plans
  3. LPAs can let us know if there are any issues with the dataset through Github discussions
  4. We continue to develop a data standard based on what we learn from this exercise
