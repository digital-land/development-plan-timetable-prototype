Development plan timetable prototype

This prototype allows you to create data that meets the development plan data standard. It allows you to record as data all the different development plans a local planning authority makes, including local plans and neighbourhood plans.

For each plan you can capture the events that happen and any documents that are produced during the plan-making process.

What should we record as a development plan?

A development plan is a set of policies and proposals that a planning authority creates to set out a vision for the future of an area. It is used to guide future planning decisions.

Record each bundle of policies a planning authority works on, that gets examined by PINs and, eventually, adopted.
